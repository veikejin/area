SMG 地区列表包
============


安装方式
-------
1.编辑composer文件包, 在require里面添加  `````"veikejin/area":"dev-master"`````

2.在repositories 里面添加
```$xslt
{
    "type": "git",
    "url": "https://github.com/veikejin/address.git",
    "reference":"master"
}
```
